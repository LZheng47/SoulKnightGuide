WHAT IS SOULKNIGHT?

SoulKnight is a fast-paced action filled mobile game that was first released in February 2017 and has over 50 million downloads. The game contains a variety of heroes with different skills that you can choose from to conquer a 3 floor dungeon filled with different enemies!
